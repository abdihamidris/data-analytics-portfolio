# data-analytics-portfolio# Electronics Sales Analysis — Power BI Dashboard

## Overview
An interactive Power BI report analyzing electronics sales 
performance across North America for January 2026. Built 
from raw Excel data through complete ETL, modeling, and 
dashboard creation.

## Business Questions Answered
- Which products generate the highest revenue and profit?
- How does performance differ between USA and Canada?
- Which sales reps drive the most value?
- What discount levels optimize profitability?

## Tools Used
- Power BI Desktop
- Power Query (M language) for data transformation
- DAX for measures and calculations
- Star schema data modeling

## Key Findings
- Total revenue: $99,430 across 100 orders
- Top product: MacBook Pro 14" ($15,540 revenue)
- Most profitable product: Samsung TV 65" ($2,000 profit)
- Alice generates 46% of total sales revenue
- 10% discount tier yields highest margin (16.1%)
